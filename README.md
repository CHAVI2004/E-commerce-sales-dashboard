# 🛒 E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard built to analyze e-commerce sales performance. It provides insights into sales, profit, quantity, customer behavior, payment methods, and state-wise performance.

## 📊 Dashboard Features
- Total Sales Amount
- Total Quantity Sold
- Total Profit
- Average Order Value (AOV)
- Monthly Profit Analysis
- Sales by State
- Sales by Customer
- Profit by Product Sub-Category
- Payment Mode Analysis
- Quarter-wise Filtering

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel

## 📈 Key Insights
- Total Sales: **161K**
- Total Profit: **26K**
- Total Quantity Sold: **2008**
- COD is the most used payment method.
- Maharashtra has the highest sales.
- Printers generate the highest profit among sub-categories.
